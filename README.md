# Skill Quest

**Skill Quest** is a simple learning roadmap web app designed to help students learn topics step by step.  
It is focused on skills like **Python, AI/ML, logic building, data science, robotics, and investing basics**.

## ⚙️ How It Works

Each learning topic is shown as a task or quest.
When the user clicks the **LEARN** button, it shows multiple learning options like:

* Google Search
* ChatGPT
* Claude
* Perplexity

Each option opens with a ready-made prompt to learn the topic faster and clearly.

## 🎯 Purpose of This Project

The main goal of this project is to create a personal learning system that helps students:

* Learn in the correct order
* Avoid confusion
* Track important skills
* Learn faster using AI tools
* Build a strong foundation for career growth
